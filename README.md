# Thus

Graphical Installer for KaOS, forked from Manjaro Linux

## Dependencies

 * coreutils
 * cryptsetup
 * gptfdisk
 * grep
 * gtk 3
 * libtimezonemap
 * hdparm
 * lvm 2
 * mhwd
 * parted (e2fsprogs, dosfstools, jfsutils, ntfs-3g, ntfsprogs, reiserfsprogs, xfsprogs)
 * pam_encfs
 * py3parted (pyparted on python3) -> https://github.com/antergos/manjaro-packages/tree/master/py3parted
 * python-configobj
 * python 3
 * python-gobject 3
 * python-dbus
 * python-cairo
 * rsync
 * sudo
 * systemd
 * keyboardctl
 * ttf-aboriginal-sans
 * ttf-indic-otf
 * ttf-khmer
 * ttf-lohit-fonts
 * ttf-myanmar3
 * ttf-thaana-fonts
 * ttf-tlwg
 * upower
 * util-linux
 * webkitgtk 3
